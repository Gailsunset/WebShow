第三步
用html实现以下作品：
3．功能
可视化展示八大行星（含地球）的椭圆轨道，标注行星名称及轨道半径比例
支持行星公转演示（按真实比例转速运行，可暂停）
提供基础参数显示：行星名称、与太阳距离（简化数值）
支持手动调节公转速度（快 / 慢 / 暂停三档）
适配桌面端（电脑 / 平板）基础显示与操作
4．交互流程
flowchart TB
    A[用户打开产品] --> B[显示初始界面：八大行星椭圆轨道+轨道起点行星+名称标注]
    B --> C{用户操作}
    C -->|点击“启动公转”| D[行星按真实比例转速沿轨道运行]
    C -->|点击“暂停”| E[公转停止，保持当前行星位置]
    C -->|点击“增速”/“减速”| F[调节行星公转速度]
    C -->|点击任意行星| G[弹出浮窗：显示行星名称+与太阳简化距离]
    C -->|点击“重置”| H[行星回归轨道起点，公转停止]
    D --> C
    E --> C
    F --> C
    G --> C
    H --> C
