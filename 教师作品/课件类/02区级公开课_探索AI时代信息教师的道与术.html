理解 Vibe Coding 的本质与价值

一、Vibe Coding 的定义

Vibe Coding（氛围编程）是一种使用AI辅助的全新编程范式。在这种模式下：

  • 程序员用自然语言描述要处理的问题
  • 将需求提供给大型语言模型（LLM）
  • AI 自动生成应用程序源代码
  • 程序员的工作从"撰写代码"转变为"指导AI、测试和优化代码"

历史背景：这一概念由安德烈·卡帕斯（Andrej Karpathy）在2025年2月提出，并在3月被收录进韦氏英语惯用法词典的"slang & trending"词条。

实操步骤

第一步：初始需求描述

做一个网页随机抽人，人名放记事本中，打开时网页再导入。
人名全部随机分布在网页中，颜色随机。抽取时，每个人名快速向上移动，
到页面顶端再从页底循环。像黑客帝国数字1或0不停向上移动效果。
抽取时，高亮放大显示被抽取到的姓名

第二步：迭代优化

  • • 人名上下流动感觉，最多十列
  • • 屏幕要一直填满，选中人名居中高亮
  • • 支持下一轮随机抽取
  • • 添加关闭按钮，开启下一轮

教学要点

  • • 需求分解：从大需求拆分成小功能
  • • 描述清晰：使用具体的视觉描述
  • • 测试反馈：记录有效的 Prompt

提升质量的六大技巧

1

行动 + 目的

明确指出要执行的动作与预期的结果

❌ 不好的 Prompt

做一个点名器

✅ 好的 Prompt

用 HTML+CSS+JavaScript 创建一个随机点名器
要求：可以从文本框输入学生名单、点击按钮随机选择
添加酷炫的动画效果、使用黑客帝国风格的绿色主题
2

互动式编码

持续对话微调，像与工程师协作

• "请优化这段循环效能"

• "改成符合 Bootstrap 的样式"

• "添加响应式布局支持"

3

添加条件限制

通过技术条件控制输出范围

• "请使用 pandas 套件实作"

• "限制在 50 行程式码内完成"

• "使用纯 CSS 实现，不用 JavaScript"

4

用注释补充

清晰的注释帮助 AI 理解逻辑意图

// 这段代码用来验证用户输入的表单数据
// 需要检查：邮箱格式、密码强度、必填项

    // 请帮我完善这个函数
}
